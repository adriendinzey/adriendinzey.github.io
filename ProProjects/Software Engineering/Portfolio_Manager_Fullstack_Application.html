This is a fullstack application with the frontend written in JavaScript using the Angular framework and a RESTful API backend written in Python using the Django framework, which connects to a MySQL database. This project utilizes the yFinance API to get real-time data from Yahoo Finance so users can buy and sell stocks based on their current prices at the time of the transaction. Users can fund their account and buy and sell stocks as time goes on, and can then review their gains and losses from week to week.
