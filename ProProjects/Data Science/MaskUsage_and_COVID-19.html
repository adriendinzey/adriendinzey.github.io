Mask Usage and COVID-19

09/2021 â€” 01/2022 (completed)

This project provided a data-based argument as to whether or not mask usage help prevent the spread of COVID-19. While there are many valid arguments using scientific thought as to why this is the case, this project gives a simpler evidence-based, data-driven reason. It examines data maintained by The New York Times on COVID-19 cases, deaths and mask usages and uses Nonparametric Statistical methods to determine the relationships present in the data.
