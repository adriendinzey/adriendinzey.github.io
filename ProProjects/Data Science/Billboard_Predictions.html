Billbaord Predictions

The goal of this project is to discover if there is a "formula" for making a hit Billboard song. The project uses data from this data base and creates a predictive model based off objective qualities of songs such as tempo, key, instrumentalness as well as qualities that are commonly seen as subjective such as valence and dancability to predict whether or not a song will make the Top 100 Billboard, as well as make a model to determine the most likely position on the Billboard. Linear Regression and Logistic Regression predictive models were created, resulting in inaccurate models. The flaws of using these techniques on this particular dataset are discussed.
